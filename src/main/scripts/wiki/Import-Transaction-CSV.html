In pachira, transactions can also be imported through .csv files. This is quite convenient for handle certain repetitive transactions.

First create a csv file with following column headings on the first row:

  • Account Name
  • Date
  • Action
  • Category
  • Payee
  • Security Name
  • Amount
  • Quantity

The remaining file would contain transactions to be imported. Any line with empty Account Name or empty Date would be skipped. Currently, only the following Action types are supported:

ActionDescription
BUY Buy shares
SELL Sale shares
DIVCash dividend
REINVDIVReinvest dividend
INTINCInterest income
REINVINTReinvest interest
CGLONGLong-term capital gain
CGMIDMid-term capital gain
CGSHORTShort-term capital gain
REINVLGReinvest long-term capital gain
REINVMDReinvest mid-term capital gain
REINVSHReinvest short-term capital gain
DEPOSITDeposit into the account
WITHDRAWWithdraw from the account

Once the csv file is ready, in Pachira, select File -> Import... -> Transactions - CSV to import the transactions.
